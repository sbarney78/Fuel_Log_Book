FuelLogBook
A precision fuel tracking and vehicle telemetry tool built for Australian drivers.

FuelLogBook helps you monitor fuel usage, track running costs, and maintain accurate vehicle records using Australian‑standard calculations. Designed for households, enthusiasts, and small business fleets, it delivers a clean, premium experience with fast data entry and reliable offline storage.

Features
Accurate Fuel & Distance Tracking
Calculates fuel economy using the Australian L/100km standard

Automatically inherits your previous odometer reading

Ensures consistent, error‑free distance and efficiency data

Supports Multiple Vehicles
Manage unlimited vehicles

Organised by Rego

Ideal for families, enthusiasts, and small business fleets

91 / 95 / 98

E10

Diesel

Premium blends

Regional or specialty fuels

Each entry records the exact fuel type used, helping you compare performance and cost over time.
